## Installation

Add this line to your application's Gemfile (currently under development):

```ruby
gem 'telegram_bot'
```

And then execute:

    $ bundle

## Deploy to Heroku

[Set up a new app](https://devcenter.heroku.com/articles/creating-apps)

And then push to it:

    $ git push heroku master

Run it:

    $ heroku ps:scale bot=1
